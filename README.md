# اختبر معلوماتك الإسلامية

لعبة أندرويد عربية لاختبار المعلومات الإسلامية بطريقة بسيطة وتفاعلية.

## محتوى اللعبة

تحتوي اللعبة على 5 أقسام، وفي كل قسم 5 أسئلة متنوعة:

- القرآن الكريم
- الأنبياء
- السيرة النبوية
- العبادات
- الصحابة

## المزايا

- 25 سؤالًا بأربع اختيارات لكل سؤال.
- ترتيب الأسئلة يتغير عند إعادة اللعب.
- توضيح الإجابة الصحيحة وشرح مختصر بعد كل سؤال.
- نظام نقاط ونتيجة نهائية بالنسبة المئوية.
- إمكانية إعادة القسم أو اختيار قسم جديد.
- واجهة عربية RTL ووضع فاتح وداكن.
- يعمل دون إنترنت.

## التقنيات

- Kotlin
- Jetpack Compose
- Material 3
- Android SDK 35
- JDK 17

## بناء التطبيق

```bash
./gradlew assembleDebug
```

ملف APK الناتج:

```text
app/build/outputs/apk/debug/app-debug.apk
```

يعمل التطبيق على Android 8.0 فأحدث.
