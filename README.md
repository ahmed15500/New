# يومياتي — Yawmeyaty

تطبيق أندرويد عربي لتتبّع المهام والعبادات اليومية بأسلوب بسيط ومحفّز.

## المزايا الحالية

- قائمة يومية جاهزة للصلوات والأذكار وورد القرآن والعمل الصالح.
- إضافة مهام إسلامية أو عادات نافعة مخصّصة.
- متابعة نسبة الإنجاز اليومية.
- سلسلة التزام للأيام التي يتجاوز فيها الإنجاز 70%.
- فلاتر للمهام: الكل، المتبقي، المكتمل.
- تخزين محلي بالكامل دون حساب أو اتصال بالإنترنت.
- واجهة عربية RTL مع الوضع الفاتح والداكن.
- عرض التاريخ الميلادي والهجري وفق تقويم أم القرى المتاح على الجهاز.

## التقنيات

- Kotlin
- Jetpack Compose
- Material 3
- SharedPreferences + JSON للتخزين المحلي
- Android Gradle Plugin 9.2.1
- Compose BOM 2026.06.00

## التشغيل

1. افتح المشروع في أحدث إصدار مستقر من Android Studio.
2. استخدم JDK 17 أو أحدث مدعوم من Android Studio.
3. ثبّت Android SDK 37 من SDK Manager.
4. نفّذ Gradle Sync ثم شغّل التطبيق على هاتف أو محاكي يعمل بنظام Android 8.0 فأحدث.

من سطر الأوامر:

```bash
./gradlew assembleDebug
```

سيظهر ملف APK التجريبي في:

```text
app/build/outputs/apk/debug/app-debug.apk
```
