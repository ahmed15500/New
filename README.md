# اختبر معلوماتك الإسلامية

لعبة أندرويد عربية لاختبار المعلومات الإسلامية، مع بنك أسئلة مركزي يمكن تحديثه بعد نشر التطبيق دون إصدار APK جديد.

## المزايا

- 5 أقسام و25 سؤالًا في الإصدار الأول.
- تحميل تحديثات الأسئلة تلقائيًا عند تشغيل التطبيق.
- زر يدوي باسم **تحديث الأسئلة الآن**.
- حفظ آخر بنك أسئلة على الهاتف للعمل بدون إنترنت.
- عرض رقم إصدار البنك وتاريخ آخر تحديث.
- أربع اختيارات وشرح الإجابة ونظام نقاط ونتيجة نهائية.
- واجهة عربية RTL ووضع فاتح وداكن.

## تحديث الأسئلة مستقبلًا

المصدر المركزي هو:

```text
question-bank/questions.json
```

لتحديث جميع المستخدمين:

1. افتح الملف على GitHub.
2. اضغط زر القلم **Edit**.
3. أضف السؤال داخل القسم المطلوب، مع `id` جديد وفريد.
4. ارفع قيمة `version` مثلًا من `1` إلى `2`.
5. حدّث `updatedAt`.
6. احفظ التعديل في فرع `main`.

عند فتح المستخدمين للتطبيق، سيتم تنزيل البنك الجديد تلقائيًا. لا يلزم إعادة بناء أو إرسال APK جديد.

راجع `QUESTION_BANK_GUIDE.md` للحصول على نموذج سؤال جاهز وقواعد التعديل.

## التقنيات

- Kotlin
- Jetpack Compose
- Material 3
- JSON + GitHub Raw Content
- تخزين محلي للعمل دون إنترنت
- Android SDK 35
- JDK 17

## بناء التطبيق

```bash
./gradlew assembleDebug
```

ملف APK الناتج:

```text
app/build/outputs/apk/debug/app-debug.apk
```

يعمل التطبيق على Android 8.0 فأحدث.
