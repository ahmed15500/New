# ميداني — Midani

تطبيق أندرويد عربي لإدارة المهام والمشروعات والزيارات الميدانية والمصروفات اليومية، مصمم للعمل بدون إنترنت.

## المزايا

- لوحة يومية تعرض المهام المتبقية والمكتملة ومصروف اليوم.
- مهام مرتبطة بالمشروع والأولوية والموعد.
- سجل للزيارات الميدانية يشمل المكان والملاحظات وعدد المستفيدين.
- تسجيل المصروفات وتصنيفها حسب المشروع.
- لوحة مشروعات تلخص نسبة الإنجاز والزيارات والمصروفات.
- مشروعات جاهزة تشمل القرى الـ13، إدارة المخلفات، Arts for Climate، المدرسة الصيفية، GWS-SENCE، وSustainability Week.
- تخزين محلي بالكامل دون حساب أو اتصال بالإنترنت.
- واجهة عربية RTL ووضع فاتح وداكن.

## التقنيات

- Kotlin
- Jetpack Compose
- Material 3
- SharedPreferences + JSON
- Android SDK 35
- JDK 17

## التشغيل

```bash
./gradlew assembleDebug
```

سيظهر ملف APK في:

```text
app/build/outputs/apk/debug/app-debug.apk
```

يعمل التطبيق على Android 8.0 فأحدث.
