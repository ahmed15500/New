# إيكو ويست — إدارة اشتراكات جمع المخلفات

تطبيق أندرويد عربي متصل بقاعدة Supabase مركزية لإدارة المشتركين في خدمة إيكو ويست.

## الصلاحيات

- **Admin:** إضافة المشتركين، تعديل البيانات، وتجديد اشتراك الشهر الحالي.
- **Field Officer:** البحث بالاسم أو الهاتف وعرض حالة الاشتراك فقط.

## منطق الاشتراك

يظهر العميل «مشترك» فقط عند وجود اشتراك في نفس الشهر والسنة الحالية. إذا كان آخر دفع في شهر سابق ولم يتم التجديد، يظهر «غير مشترك» تلقائيًا.

## البيانات

تم استيراد 302 سجلًا من ملفي المشتركين وغير المشتركين، بينها 246 اشتراكًا لشهر يوليو 2026.

## التقنية

Kotlin، Jetpack Compose، Material 3، Supabase Auth، PostgREST وRLS.
