# دليل تحديث بنك الأسئلة

ملف بنك الأسئلة المركزي:

```text
question-bank/questions.json
```

## إضافة سؤال إلى قسم موجود

أضف هذا الكائن داخل قائمة `questions` الخاصة بالقسم المطلوب:

```json
{
  "id": "quran-006",
  "question": "اكتب السؤال هنا",
  "options": [
    "الاختيار الأول",
    "الاختيار الثاني",
    "الاختيار الثالث",
    "الاختيار الرابع"
  ],
  "correctAnswer": 1,
  "explanation": "شرح مختصر للإجابة الصحيحة"
}
```

`correctAnswer` يبدأ من الصفر:

- `0` = الاختيار الأول
- `1` = الاختيار الثاني
- `2` = الاختيار الثالث
- `3` = الاختيار الرابع

## إضافة قسم جديد

أضف كائنًا جديدًا داخل قائمة `categories`:

```json
{
  "id": "hadith",
  "title": "الحديث الشريف",
  "emoji": "📜",
  "description": "أسئلة في الحديث النبوي",
  "questions": [
    {
      "id": "hadith-001",
      "question": "اكتب السؤال هنا",
      "options": ["أ", "ب", "ج", "د"],
      "correctAnswer": 0,
      "explanation": "شرح الإجابة"
    }
  ]
}
```

## قواعد مهمة

1. يجب أن يحتوي كل سؤال على أربعة اختيارات بالضبط.
2. يجب أن يكون `id` فريدًا ولا يتكرر.
3. بعد كل تحديث، ارفع قيمة `version` بمقدار واحد.
4. حدّث قيمة `updatedAt` إلى تاريخ التعديل.
5. تأكد من صحة الفواصل والأقواس في JSON قبل الحفظ.
6. الأسئلة الدينية يجب مراجعتها من مصدر موثوق قبل النشر.

بعد الحفظ على فرع `main`، سيصل التحديث للمستخدمين تلقائيًا عند فتح التطبيق أو الضغط على **تحديث الأسئلة الآن**.
